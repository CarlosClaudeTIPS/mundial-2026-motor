# Motor Mundial 2026

Motor de datos y estadísticas para el Mundial de Fútbol 2026.

## Descripción

Este proyecto centraliza información sobre equipos, grupos, partidos y estadísticas del Mundial 2026 (Estados Unidos, Canadá y México).

## Características

- Seguimiento de grupos y clasificaciones
- Estadísticas de equipos y jugadores
- Resultados de partidos en tiempo real

## Instalación

```bash
npm install
```

## Uso

```bash
npm start
```
